# Üzenőfal App

A simple message board web application built with Next.js, Supabase, and Vercel.

## Features
- Add messages
- View messages in reverse chronological order
- Delete messages

## Tech Stack
- Next.js (React, TypeScript)
- Supabase (Database)
- Tailwind CSS (Styling)
- Vercel (Deployment)

## Running locally
npm install  
npm run dev

## Notes
This project was developed with the assistance of AI tools (ChatGPT) to speed up development, debug issues, and structure the application.
